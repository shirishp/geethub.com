# Geethub.com

A modern, lightweight typo page that helps users who accidentally type "geethub.com" instead of "github.com". Built with pure HTML/CSS for maximum performance and deployed on GitHub Pages.

## 🚀 Deployment

This site is deployed using **GitHub Pages**:

1. **Repository Setup**: Push code to the `master` branch
2. **GitHub Pages**: Automatically enabled for repository
3. **Custom Domain**: Configured via CNAME file
4. **HTTPS**: Automatically enabled by GitHub Pages
